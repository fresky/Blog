推荐一个Visual Studio的插件[CodeMaid](http://www.codemaid.net/)，它有很多的功能，比如可以自动清理代码，帮助我们浏览代码。

我最喜欢的功能是直接在Visual Studio中看到complexity，这样在重构时就可以有的放矢，避免老打开[SourceMonitor](http://www.campwoodsw.com/sourcemonitor.html)了。
