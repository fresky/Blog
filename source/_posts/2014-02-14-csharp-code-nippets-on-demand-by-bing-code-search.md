[Bing Code Search: C# code snippets on-demand](http://codesnippet.research.microsoft.com/)可以用三个斜杠///作为注释，然后按下tab，就能用bing的code search搜索相关的代码片段，直接回车插入到进来。
主流IDE的插件正在开发，可以在网页中先试试，挺好用的。
